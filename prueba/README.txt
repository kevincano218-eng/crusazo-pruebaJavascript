El proyecto simula un flujo completo de gestión de tareas usando una API falsa con JSON Server y una interfaz basada en diseños de Figma.Tecnologías

HTML5, CSS3
JavaScript (Vanilla)
Bootstrap 5, Materialize, Foundation, Bulma, Tailwind CSS
JSON Server (API falsa)
LocalStorage / SessionStorage (persistencia de sesión)

Módulo Usuario

Registro de cuenta nueva (rol user automático), Login y persistencia de sesión, Gestión de tareas: crear, editar, eliminar y listar tareas, Cambio de estado de tareas: pending, in progress, completed.
Perfil de usuario y cierre de sesión

Módulo Administrador

Login con redirección según rol
Dashboard con métricas: total de tareas, pendientes, completadas
Gestión completa de tareas: ver, editar, eliminar y cambiar estado
Gestión de usuarios (opcional)
